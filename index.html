Sudoku

El sudoku es un juego entretinido creado por el matematico Leonhard Euler, se trata de un sistema de probabilidades para representar una serie de nuemero sin repetir inicalmente el nombre tiene que ver con japon donde una editorial publico una seccion de pasatiempos llamada Sūji wa dokushin ni kagiru. El nombre se abrevió a Sūdoku (sū = número, doku = solo). Este juego se ve comunmente en una tabla de 9× 9, compuesta por pequeñas sub tablas de 3 × 3 denominadas "regiones" algunas celdas contienen numeros estas son las pistas, el objetivo es rellenar las celdas vacías, con un número end cada una de ellas, de manera que cada columna, fila y región contenga los números 1–9 solo una vez es decir sin repetirse.
